# SharedPrefernce
SharedPreferences is an API from Android SDK to store and retrieve application preferences.
SharedPreferences are simply sets of data values that stored persistently. Persistently which mean data you stored in the SharedPreferences 
are still exist even if you stop the application or turn off the device.
